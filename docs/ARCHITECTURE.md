# Architecture

Argobrain Memory separates evidence from views.

## Authority Order

1. Raw transcript or event log.
2. Append-only compaction episodes.
3. Typed fact ledger.
4. Active derived summary.
5. Recall index.

The lower layers are evidence. The upper layers are conveniences.

## Borrowed Patterns

- GBrain: schema discipline, doctor/remediation loops, explicit user gates.
- Mem0: memory operation evaluation and auto-capture caution.
- Letta/MemGPT: working memory vs archival memory vocabulary.
- Graphiti/Zep: temporal facts, entity lifecycle, contradiction/supersession ideas.
- codebase-memory-mcp: rebuildable local index and read-contract patterns.
- Event sourcing/CQRS: append-only source records and rebuildable views.

No external memory runtime is required by this package.
